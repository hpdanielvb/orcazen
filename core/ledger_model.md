# Ledger Model

## Estrutura

- journal_entries
- journal_lines

## Regras Contábeis

- Toda transação gera um lançamento
- Todo lançamento possui pelo menos duas linhas
- Soma de débitos e créditos = 0

## Convenção

- Débito → positivo
- Crédito → negativo

## Objetivo

Garantir consistência financeira absoluta.
